# {News} In Brief

Generate an informative daily audio briefing from selected RSS feeds.

</div>

## Listen

Listen to the show's episodes by adding the RSS URL to your favorite Podcast player (like [AntennaPod](https://antennapod.org/)) 

<div align="left" style="vertical-align: middle;">

![Rss](https://img.shields.io/badge/rss-F88900?logo=rss&logoColor=white) 
`https://p1va.github.io/news-in-brief/rss.xml`
</div>

## Installation

```sh
uv sync
```

## Run

Before you run make sure to have `OPENROUTER_API_KEY` and `ELEVENLABS_API_KEY` in your environment then

```sh
uv run main.py
```
